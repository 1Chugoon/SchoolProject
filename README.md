# Учебный проект: Сайт курсов

## Описание
Этот проект представляет собой учебный веб-сайт, где пользователи могут создавать и приобретать курсы. Основная цель — закрепление навыков разработки веб-приложений и работы с современными технологиями.

## Цели проекта
- Научиться создавать интерактивные веб-приложения.  
- Освоить работу с базой данных через Entity Framework.  
- Реализовать аутентификацию и авторизацию с использованием JWT.  
- Развить навыки работы с фронтенд-технологиями (React, CSS, JavaScript).  
- Настроить приложение в контейнере Docker для удобного развертывания.

## Технологии и языки
- **Языки:**  
  - C# 
  - JavaScript 
  - CSS
  - HTML
- **Технологии:** React, Entity Framework, .NET, JWT, Docker и другие для .NET-экосистемы.


## Примеры использования

Создание нового курса.

Просмотр доступных курсов и покупка выбранных.

Авторизация пользователей с JWT-токеном.

## Структура проекта

/backend – серверная часть на .NET с EF.

/frontend – клиентская часть на React.

/docker – конфигурация для контейнеризации приложения.
